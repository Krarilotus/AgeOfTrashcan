Tower SVG Assets

Files:
- `player_level_1.svg` ... `player_level_10.svg`
- `enemy_level_1.svg` ... `enemy_level_10.svg`

Naming:
- `{side}_level_{N}.svg`
- `side`: `player` or `enemy`
- `N`: turret level

Note:
- Runtime currently uses original procedural tower rendering for exact legacy parity.
- These SVGs are the organized tower asset set for future switch-over.
